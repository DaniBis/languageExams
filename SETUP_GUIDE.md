### Common Issues

**Issue:** Fonts not loading
**Solution:** Check font imports in `app/layout.tsx` and CSS variables in `tailwind.config.ts`

**Solution:** Ensure `globals.css` includes all Shadcn CSS variables

**Issue:** TypeScript errors
**Solution:** Run `npm install --save-dev @types/node @types/react @types/react-dom`

**Issue:** Build fails
**Solution:** Check for missing dependencies: `npm install`

**Issue:** Contact form not working
**Solution:** Implement email sending in `app/api/contact/route.ts`
